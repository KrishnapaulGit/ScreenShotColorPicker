📚 API Examples

Step 1: Generate Token (cURL) 
curl -X POST http://localhost:5000/api/generate-token \
  -H "Content-Type: application/json" \
  -d '{
    "name": "John Doe",
    "email": "john@example.com"
  }'
Step 2: Process Image with Token (cURL) 
curl -X POST http://localhost:5000/api/piccolour \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -H "Content-Type: application/json" \
  -d '{
    "imageData": "data:image/png;base64,iVBORw0KGgoAAAANS..."
  }'
JavaScript/Fetch Example 
// Generate token
const tokenRes = await fetch('/api/generate-token', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ name: 'John', email: 'john@example.com' })
});
const { token } = await tokenRes.json();

// Process image
const imageRes = await fetch('/api/piccolour', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },

});
